Lista de Exercícios 01 — Programação Orientada a Objetos (OO)

Este projeto contém uma sequência de exercícios práticos em JavaScript, focados em Programação Orientada a Objetos (POO), indo do nível básico ao avançado.

O objetivo é desenvolver raciocínio lógico, domínio de classes, objetos, herança, polimorfismo e composição, além de integração com interface web (HTML, CSS e JavaScript).

Tecnologias utilizadas
HTML5
CSS3
JavaScript (ES6+)
Programação Orientada a Objetos (POO)
Estrutura dos exercícios

Filme (verificação de tempo de lançamento)

Objetivo do projeto

Este projeto foi desenvolvido para treinar:

Pensamento orientado a objetos
Estruturação de sistemas reais
Manipulação de múltiplos objetos
Lógica de negócios
Integração com interface web
O que você aprende com isso

Ao concluir todos os exercícios, você será capaz de:

Criar sistemas completos em JavaScript
Modelar problemas do mundo real com classes
Usar herança e polimorfismo corretamente
Construir aplicações web interativas
Evoluir para frameworks como React ou Node.js
